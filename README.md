# 🎮 GameVault

A game discovery app built with React — browse, search, and save your favorite games.

## Features

- 🔍 Search games by name
- 🎯 Filter by genre
- 📄 Game detail page
- ❤️ Wishlist with localStorage persistence

## Tech Stack

- React + Vite
- React Router v6
- Tailwind CSS
- RAWG API

## Getting Started

1. Clone the repo
2. Install dependencies
   \`\`\`bash
   npm install
   \`\`\`
3. Create `.env` and add your RAWG API key
   \`\`\`
   VITE_RAWG_API_KEY=your_api_key_here
   \`\`\`
4. Run the app
   \`\`\`bash
   npm run dev
   \`\`\`
